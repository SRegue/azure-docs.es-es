# <a name="azure-storage-partners-for-primary-and-secondary-storage"></a>Asociados de Azure Storage para el almacenamiento principal y secundario

En este artículo se destacan las empresas asociadas de Microsoft que ofrecen una solución de almacenamiento conectado a la red (NAS) o de red de área de almacenamiento (SAN). La solución puede ser local o estar hospedada en Azure, o bien tratarse de una solución híbrida que use Azure Storage como una opción de nivel rentable. Estas soluciones pueden permitir a los clientes usar la misma solución en cualquiera de sus entornos.
